# Movie Recommendation System

## Overview
This project is a content-based movie recommendation system that suggests similar movies based on user input. It analyzes features like genres, keywords, cast, and crew, and uses cosine similarity to recommend the top 5 most relevant movies. The system is deployed as an interactive Streamlit web application.

## Live Demo
https://movie-recommendation-system-krish.streamlit.app/


## Features
- Content-based filtering  
- Top-5 similar movie recommendations  
- Uses cosine similarity for accurate suggestions  
- Built using Pandas and Scikit-learn    

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit
